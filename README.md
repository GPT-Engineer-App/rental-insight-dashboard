# rental-insight-dashboard

A dashboard that a potential tenant could view to gain insights into a potential rental home. The dashboard should show things like the homes estimated market value rent, price trends in the area, neighborhood insights, number of view, applications and favorites the home has, the homes price drops or increases over time, likelihood of being taken off the market soon, etc.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/rental-insight-dashboard.git
cd rental-insight-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
